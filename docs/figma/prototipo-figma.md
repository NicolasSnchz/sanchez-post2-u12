# Prototipo Figma - NotesSync Integrador

## Flujo principal requerido

El prototipo debe tener minimo 5 pantallas navegables:

1. Splash / Bienvenida
2. Login
3. Lista de notas
4. Crear nota
5. Detalle de nota
6. Estado offline / sincronizacion

## Design System minimo

### Colores

- Primary: #1E3A8A
- Secondary: #2563EB
- Background: #F8FAFC
- Error: #DC2626
- On Primary: #FFFFFF

### Tipografia

- Headline: 28sp / Bold
- Title: 20sp / SemiBold
- Body: 16sp / Regular
- Label: 14sp / Medium

### Componentes reutilizables

- Button
- TextField
- Card
- TopAppBar
- SyncStatusChip

### Estados

- Default
- Focused
- Error
- Disabled

## Navegacion sugerida

Splash -> Login -> Lista de notas -> Crear nota -> Detalle nota -> Estado offline/sync
